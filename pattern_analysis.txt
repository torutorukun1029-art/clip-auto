# StockSun YouTubeショート動画 再生数分析レポート

## 1. 伸びている動画の共通パターン

### 【タイトルの型】TOP5パターン

```
■ パターン①：数字インパクト型（最強）
「年収は3億円！」「44万円の自転車」「月給30万」
→ 具体的な金額が入ると圧倒的に強い

■ パターン②：感情爆発型
「ブチギレ」「不満爆発」「嬉し泣き」
→ 感情の振れ幅が大きいワードが刺さる

■ パターン③：ネガティブ暴露型
「本当にあった酷い〜」「最低な〜」「絶対に〜したくない」
→ 業界の闘い話・失敗談は鉄板

■ パターン④：疑問形・質問型
「〜はいくらかかるの？」「〜の違いは？」「〜とは！？」
→ 視聴者の知りたい欲求を刺激

■ パターン⑤：人物フィーチャー型
「株本が〜」「岩野と株本の〜」「金子が〜」
→ キャラクター名＋行動で興味喚起
```

## 3. 切り抜きで伸ばすための具体的指針

### 🎯 最優先で狙うべきシーン

```
【即採用レベル】
□ 具体的な金額が出る瞬間
  「〜は◯◯万円です」「年収は〜」

□ 誰かが怒る/怒られるシーン
  声のトーンが変わる、空気が張り詰める瞬間

□ 業界の裏話・ぶっちゃけ話
  「実は〜」「本当は〜」が出たら即メモ

□ 失敗談・やらかしエピソード
  入社初日、独立初期など「過去の恥ずかしい話」
```

### ✍️ タイトル作成テンプレート

```
【鉄板構文】
① [人名]が[感情動詞]！[具体的な状況]
   例：株本ブチギレ！岩野の発言がヤバすぎた

② [数字]万円の[対象]を[動詞]方法
   例：44万円の自転車を売るマーケティング方法

③ 絶対に[ネガティブ動詞]たくない[対象]の特徴
   例：絶対に依頼したくない制作会社の特徴

④ 本当にあった[ネガティブ形容詞][名詞]の話
   例：本当にあった酷いSEO業者の話

⑤ [人名]の[意外性のある名詞]が[形容詞]すぎる
   例：岩野の美意識が低すぎる
```

## 4. 避けるべきテーマ・切り口

### ❌ 伸びにくいパターン

| NG要素 | 理由 | 改善案 |
|:---|:---|:---|
| タイトルが説明的すぎる | 「〜の方法」だけでは弱い | 数字 or 感情ワードを追加 |
| 人物名がない | 誰の話か分からない | 必ず主語（人名）を入れる |
| ポジティブすぎる | 刺激が弱い | ギャップや葛藤を見せる |
| 専門用語が前面 | 一般層に刺さらない | 「〜したら〇〇万円損する」等に変換 |

### 📉 データから見る「避けるべき」例

```
× 「解約されないコンサルの特徴」→ 7,771回
  ↓ 改善案
○ 「99%のコンサルが3ヶ月でクビになる理由」

× 「YouTubeコンサルに成果の出し方を聞いてみた」→ 7,505回
  ↓ 改善案
○ 「登録者0→10万人にしたYouTubeコンサルの裏技」
```

```

3つ以上✓ → 優先的に切り抜く
1つ以下 → 見送りor別のシーンを探す
```
